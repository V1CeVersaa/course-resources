# CS106L 

CS106L 的难度并不算大，C++ 的各个方面均有涉及，但作为一门 C++ 基础课程，许多内容并没有深入讨论，因此难度不大。笔者在系统自学 C++ 之后，只花了两天半就把这门课的 slides 和 assignments 都过了一遍。总的来说很适合 C++ 初学者或是希望通过一些简单的作业来巩固 C++ 基础的同学。

- 课程官方网站为：[CS106L](http://web.stanford.edu/class/cs106l/)
- 课程 assignment 的 GitHub 仓库为：[CS106L](https://github.com/cs106l/cs106l-assignments)

> [!TIP]
> 本仓库中不含未经修改的原始 assignment 文件，仅包含笔者的解答。
>
> 若希望获取原始 assignment 文件，请考虑从其他学习者的仓库中获取，或直接访问官方仓库过去的 commit 记录查找。
>
